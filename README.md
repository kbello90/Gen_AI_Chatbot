# Gen_AI_Chatbot
Retrieval-Augmented Generation (RAG) system built with IBM watsonx.ai, LangChain, Streamlit, and Chroma DB  This application allows researchers to upload any scientific paper (PDF), and the AI assistant will read, understand, chunk, embed, index, retrieve, and answer questions grounded in the actual document.
